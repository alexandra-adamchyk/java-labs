# Комплексна Java програма

Програма демонструє три важливі концепції Java:
1. **Reflection** - зміна приватного поля `value` в об'єктах String
2. **Logging** - логування з різними рівнями (INFO, WARNING, SEVERE)
3. **Internationalization (I18n)** - підтримка української та англійської мов

## Файли програми

- `ComplexJavaProgram.java` - основний клас програми
- `messages_uk.properties` - ресурси українською мовою
- `messages_en.properties` - ресурси англійською мовою
- `app.log` - файл логів (створюється автоматично)

## Компіляція та запуск

```bash
# Компіляція
javac ComplexJavaProgram.java

# Запуск
java ComplexJavaProgram
```

## Функціонал

### 1. Вибір мови
При запуску програма пропонує вибрати мову інтерфейсу:
- 1 - Українська
- 2 - Англійська

### 2. Reflection приклад
Демонструє зміну приватного поля `value` в String об'єктах:
- Рядок створений як літерал
- Рядок створений через Scanner
- Вивід рядків до та після зміни

### 3. Logging приклад
Показує роботу логування з різними рівнями:
- INFO - інформаційні повідомлення
- WARNING - попередження
- SEVERE - серйозні помилки

Логи записуються як в консоль, так і в файл `app.log`.

## Особливості

- Програма адаптована для роботи з різними версіями Java
- Використовується `java.util.logging` для логування
- Підтримка інтернаціоналізації через `ResourceBundle`
- Обробка помилок при роботі з reflection

## Вимоги

- Java 8 або вище
- Для коректної роботи reflection рекомендується Java 8-17
- На новіших версіях Java може не працювати зміна поля `value` в String
