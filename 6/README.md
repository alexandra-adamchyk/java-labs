# Червоно-чорне дерево (Red-Black Tree)

Консольна програма на Java, яка реалізує структуру даних "Червоно-чорне дерево" з повним функціоналом.

## Функціонал

### Основні операції:
- **Додавання вузла** - вставка елемента з автоматичним балансуванням
- **Видалення вузла** - видалення елемента з збереженням властивостей червонно-чорного дерева
- **In-order обхід** - вивід елементів у відсортованому порядку
- **Візуалізація дерева** - текстове представлення ієрархії з кольорами вузлів

### Демонстраційні режими:
- **Випадковий масив** - тестування на випадкових числах
- **Впорядкований масив** - тестування на відсортованих числах

## Компіляція та запуск

### Компіляція:
```bash
javac RedBlackTree.java RedBlackTreeDemo.java
```

### Запуск:
```bash
java RedBlackTreeDemo
```

## Меню програми

Після запуску програми ви побачите інтерактивне меню:

1. **Додати вузол** - введіть число для додавання до дерева
2. **Видалити вузол** - введіть число для видалення з дерева
3. **Показати in-order обхід** - вивід елементів у відсортованому порядку
4. **Візуалізувати дерево** - текстова візуалізація структури дерева
5. **Демонстрація на випадковому масиві** - автоматичне тестування
6. **Демонстрація на впорядкованому масиві** - автоматичне тестування
7. **Очистити дерево** - видалення всіх елементів
8. **Вихід** - завершення програми

## Приклад візуалізації

```
Візуалізація червонно-чорного дерева:
R - червоний вузол, B - чорний вузол
└─50(B)
  ├─30(R)
  │ ├─20(B)
  │ └─40(B)
  └─70(R)
    ├─60(B)
    └─80(B)
```

## Структура файлів

- `RedBlackTree.java` - основний клас реалізації червонно-чорного дерева
- `RedBlackTreeDemo.java` - головний клас з інтерфейсом користувача

## Алгоритмічна складність

- **Додавання:** O(log n)
- **Видалення:** O(log n)
- **Пошук:** O(log n)
- **Обхід:** O(n)

## Властивості червонно-чорного дерева

1. Кожен вузол має колір (червоний або чорний)
2. Корінь завжди чорний
3. Усі NIL вузли (листя) чорні
4. Червоний вузол не може мати червоних нащадків
5. Будь-який шлях від кореня до листя містить однакову кількість чорних вузлів

## Особливості реалізації

- Використання NIL-вузлів для спрощення алгоритмів
- Автоматичне балансування при вставці та видаленні
- Інтерактивний інтерфейс з валідацією введення
- Наглядна візуалізація структури дерева
